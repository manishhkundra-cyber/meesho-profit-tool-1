This is a minimal, beginner-friendly Next.js starter with Tailwind for the Meesho profit calculator.

## What is included
- Upload & parse CSV (PapaParse)
- Column mapping UI
- Profit calculation logic
- XLSX export (SheetJS)
- Razorpay server endpoints (create-order + verify-payment)
- Tailwind-styled UI

## Quick start (local)
1. `npm install`
2. Copy `.env.example` -> `.env.local` and add values
3. `npm run dev`
4. Open http://localhost:3000

## Deploy
Push to GitHub and import into Vercel. Add env variables in Vercel dashboard before deploying.
